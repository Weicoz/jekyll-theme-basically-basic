### JS实现页面跳转的几种方式

1. window.location.href
```js
window.location.href="jb51.jsp?backurl="+window.location.href;
```
2. window.history.back
```js
window.history.back(-1); //返回上一页
```
3. window.navigate
```js
window.navigate("jb51.jsp"); 
```
4. self.location
```js
self.location='jb51.htm'; 
```
5. top.location
```js
top.location='jb51.jsp'; 
```
